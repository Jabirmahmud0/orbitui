# Installation

Install OrbitUI alongside React:

```bash
npm install @jabir0/orbitui-react react react-dom
```

## Import styles

Import the library stylesheet once near the root of your application:

```jsx
import '@jabir0/orbitui-react/dist/orbitui.css';
```

## Peer requirements

- `react >= 18.2.0`
- `react-dom >= 18.2.0`

## Local development

For this repository, the main workflows are:

```bash
npm install
npm run dev
npm run test
npm run storybook
```
